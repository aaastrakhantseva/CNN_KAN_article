# STL10 CNN + KAN: Сравнение традиционного MLP и Kolmogorov–Arnold Network

Исследовательский проект по сравнению классического полносвязного классификатора (MLP) и перспективной архитектуры **Kolmogorov–Arnold Network (KAN)** на датасете STL-10.

## О проекте

Цель проекта — исследовать эффективность замены традиционного MLP-head на KAN-head при использовании одного и того же CNN-backbone для задачи классификации изображений.

### Основные компоненты
- CNN Backbone — извлечение признаков
- MLP Head — baseline решение
- KAN Head — предлагаемое решение
- Сравнительный анализ по качеству, параметрам и интерпретируемости

## Результаты

| Метрика              | CNN + MLP   | CNN + KAN   | Изменение    |
|----------------------|-------------|-------------|--------------|
| Accuracy             | 0.6275      | **0.6601**  | +3.26%       |
| Precision            | 0.6518      | **0.6568**  | +0.50%       |
| Recall               | 0.6275      | **0.6601**  | +3.26%       |
| F1-score             | 0.6272      | **0.6574**  | +3.02%       |
| Train Loss (финал)   | 0.8799      | **0.206**   | Значительное улучшение |

**Вывод:** KAN-head демонстрирует преимущество по всем ключевым метрикам качества.
